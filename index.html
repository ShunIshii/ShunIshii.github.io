学歴

日本大学藤沢高等学校 2013.4 - 2016.3

普通クラス理系コースで学年1位の成績。サッカー部に所属し、2年次に冬の全国大会である全国高校サッカー選手権に出場、全国第三位。3年次にはポジションリーダーとして部員をまとめ、関東大会優勝、全国高校総合体育大会（インターハイ）ベスト16。防災委員会の委員長も務めるなど、文武両道が認められ卒業時に表彰。模範生として1日の生活や勉強方法などのインタビュー記事が受験生向けパンフレットで取り上げられた。

青山学院大学 2016.4 - 2020.3

理工学部情報テクノロジー学科にて工学の学士号を取得。成績優秀者（上位15%）として大学院での給付型奨学金を獲得。ウェアラブル環境情報システム研究室に所属し、卒業研究発表会では優秀賞（上位6名）を受賞。国内学会で優秀プレゼンテーション賞（上位3名を除く上位約15名）を受賞。体育会サッカー部に所属し、3・4年次にはゲームキャプテンを務めた。

青山学院大学大学院 2020.4 - 2022.3

理工学研究科理工学専攻知能情報コースにて工学の修士号を取得予定。ウェアラブル環境情報システム研究室に所属し、離れた場所から運動を共有する研究を行っている。

インターンシップ

アクセンチュア株式会社 2019.8

ある課題に対し、データ分析、戦略、プログラムの実装をチームで行った。詳細は非公開。

ヤフー株式会社 2019.8

広告コースに参加。YAHOO!JAPAN トップページに掲載する広告を想定し、よりクリックされやすい広告を抽出するアルゴリズを3人チームで実装、他チームに対し良いスコアを記録した。全文検索エンジンである Elasticsearch と Python を使用。広告配信システムの基本とチーム開発の方法を学んだ。

Turku Game Lab 2019.9 - 2019.12

研究開発インターンとして、フィンランドのトゥルク応用工科大学内の研究所でウェアラブルデバイスを用いたゲームを開発。詳細はプロジェクト欄を参照。

Apex株式会社 2020.3 - 長期継続中

外資系のハイクラスを対象とした人材紹介会社で、唯一のエンジニアとして社内業務に携わっている。ホームページの改善、約1万のデータをウェブからスクレイピング、独自の文字列置換アルゴリズムを使用してデータの整理などを行っている。また、自ら働きかけ、来客システムを電話からQRコードへ移行している。QRコードの来客システムは0から自分一人で実装。Ruby on Rails でQRコードを生成しメールで受信できる Webアプリケーションを作成、受付に設置するQRコードリーダーは Unity を用いて Androidタブレットのアプリケーションとして実装した。
